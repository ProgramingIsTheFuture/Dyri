Module Ast.TypedAst

AST with types

type typ =
| TCustom of string
| TInt
| TString
| TBool
| TGeneric

After the typing stage every variable will have one of these types

type value =
| VInt of int
| VString of string
| VBool of bool
type param = string * typ
type expr =
| Const of value
| Var of string
| Apply of string * expr list
type stmt =
| Expr of expr
| Let of string * typ * stmt
| Fun of string * typ * param list * stmt
| AnFun of param list * stmt
| If
| For
| Loop
| Block of stmt list

Possible statements to use inside the Dyri language

type code = stmt list
